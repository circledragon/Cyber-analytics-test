Answer for part B is available in "Part B Cyber threat analysis.pdf"

For part A and B, both were developed in python3.7 using anaconda. Details can be viewed in their respective folders. Both part A and B use the same environment which can be setup as follows:
```
conda create -n <environment name> python=3.7
conda activate <environment name>
pip install -r requirements.txt #or you can use pip3 instead depending on how your python is setup
```
For part A, the code requires drivers for selenium to work. They need to be downloaded and placed into your *PATH*. See the this for more information and download links for the drivers:
https://selenium-python.readthedocs.io/installation.html#downloading-python-bindings-for-selenium <br>
This link explains how to find and/or change the *PATH* for various OS: https://www.java.com/en/download/help/path.xml
